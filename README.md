# Personal AI OS Kernel

> 个人 AI 操作系统内核层 — 构建在 Hermes Agent Runtime 之上的个人智能操作系统核心层。

[![PRD](https://img.shields.io/badge/PRD-v1.0-blue)](docs/PRD.md)

## 定位

Personal AI OS Kernel 不是替代 Hermes，而是 **扩展** Hermes。

让 Hermes 从 **AI 执行助手** 升级为 **AI 个人操作系统**——拥有持续存在的自我状态、理解用户长期目标、管理个人知识、参与决策、协调 AI 团队、推动项目执行。

## 架构

```
Personal AI OS Kernel
│
Hermes Kernel
│
Tools / Skills / Agents / APIs
```
